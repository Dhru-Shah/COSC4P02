Instructions

Please, move closer to objects to hover/select using mouse.

Touchscreen users can touch the objects to select.

Click "Exit" button to exit and navigate to different pages.

Controls

Keyboard/Mouse

  1. Mouse to rotate any direction.
  2. W / Up Arrow to move forward.
  3. S / Down Arrow to move backward.
  4. A / Left Arrow to rotate/move left.
  5. D / Right Arrow to rotate/move right.

Touchscreen

  1. Joystick to move around.
  2. Swipe screen to rotate.

Virtual Reality (VR)

To experience in Virtual Reality, please put on a VR headset!
